# 🛒 TechVenger Website

A multi-page tech retail website for **TechVenger** – a modern electronics brand selling audio gear, laptops, phones, and accessories.

Built as a Semester 1 Web Development project using **HTML, CSS, and JavaScript**.

## Pages

- **Home** (`index.html`) – Hero section with image slider, product showcase, Christmas countdown timer, and footer
- **About Us** (`about.html`) – Brand story, achievements counter, and founder profiles
- **Products** (`products.html`) – Product catalog with cards and search functionality
- **Contact Us** (`contact us.html`) – Contact info and inquiry form

## Features

- Responsive navigation bar with active page highlighting
- Image slider on homepage
- Animated achievement counters (About page)
- Product search/filter
- Contact form
- Social media links
- Countdown timer for promotions

## File Structure

| File | Description |
|------|-------------|
| `index.html` | Home page |
| `about.html` | About Us page |
| `products.html` | Products page |
| `contact us.html` | Contact page |
| `index.css` | Home page styles |
| `about.css` | Shared styles (About, Products, Contact) |
| `products.css` | Product-specific styles |
| `index.js` | Home page scripts |
| `about.js` | About page scripts (counters) |

### Assets

Includes product images (`.png`/`.jpg`) for laptops, phones, watches, headphones, tablets, earbuds, and founder photos.

## How to Run

Simply open `index.html` in any modern web browser.
