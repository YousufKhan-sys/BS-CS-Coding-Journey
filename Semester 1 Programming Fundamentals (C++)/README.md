# Semester 1 — Programming Fundamentals (C++)

A collection of **67 C++ programming tasks** completed during the first semester of a BS Computer Science program. The assignments progress from basic syntax and variables through pointers and file handling, building a strong foundation in procedural programming with C++.

## How to Compile & Run

Compile any task with `g++` (or any C++ compiler) and run the resulting executable:

```bash
g++ Task01_MilesToKilometers.cpp -o Task01
./Task01
```
