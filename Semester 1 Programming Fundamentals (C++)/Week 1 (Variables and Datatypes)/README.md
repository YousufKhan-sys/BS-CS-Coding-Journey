# Week 1 — Variables and Datatypes

Introduction to C++ variables, fundamental data types, and basic I/O operations.

## Tasks

| # | File | Description |
|---|------|-------------|
| 1 | `Task01_MilesToKilometers.cpp` | Converts a distance from miles to kilometers |
| 2 | `Task02_WithdrawFromAccount.cpp` | Simulates a bank withdrawal with balance validation |
| 3 | `Task03_DayOfWeekCalculator.cpp` | Determines the day of the week for a given date |
| 4 | `Task04_PrintLetterY.cpp` | Prints the letter Y using output formatting |
| 5 | `Task05_ServiceInvoiceCalculator.cpp` | Calculates a service invoice total |
| 6 | `Task06_SecondsToTimeFormat.cpp` | Converts seconds into HH:MM:SS format |
| 7 | `Task07_NumberGuessingGame.cpp` | Simple number guessing game |

## Concepts Covered

- Variable declaration and initialization
- `int`, `float`, `double`, `char` data types
- `cin` / `cout` for input and output
- Arithmetic operators and type conversion
- Basic problem decomposition
