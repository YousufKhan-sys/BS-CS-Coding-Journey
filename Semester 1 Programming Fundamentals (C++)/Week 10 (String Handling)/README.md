# Week 10 — String Handling

Working with C++ `string` objects for text manipulation and processing.

## Tasks

| # | File | Description |

## Concepts Covered

- `std::string` class
- String concatenation and comparison
- `length()`, `substr()`, `find()`
- String modification (`insert`, `erase`, `replace`)
- Character traversal and case conversion
- C-string vs C++ string differences
