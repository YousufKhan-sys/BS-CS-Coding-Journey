# Week 11 — File Handling

Reading from and writing to external files using `fstream`, `ifstream`, and `ofstream`.

## Tasks

| # | File | Description |
|---|------|-------------|
| 59 | `Task59_FileBackup.cpp` | Creates a backup copy of a text file |
| 60 | `Task60_QuoteSaver.cpp` | Saves user-entered quotes to a file |
| 61 | `Task61_StudentRecordManagement.cpp` | Reads/writes student records from/to a file |
| 62 | `Task62_GradeUpdater.cpp` | Reads grades from a file, updates them, and saves |

## Concepts Covered

- `ofstream` for file output
- `ifstream` for file input
- `fstream` for read/write
- File open / close / error checking
- Reading lines with `getline()`
- Appending and overwriting file content
- Persistent data storage between program runs
