# Week 2 — Conditional Statements

Applying `if`, `if-else`, and `switch` statements to build decision-making programs.

## Tasks

| # | File | Description |
|---|------|-------------|
| 8 | `Task08_CharacterClassification.cpp` | Classifies a character as letter, digit, or symbol |
| 9 | `Task09_ShoppingCart.cpp` | Calculates total with discounts based on cart value |

## Concepts Covered

- `if`, `else if`, `else` branching
- `switch` / `case` statements
- Logical operators (`&&`, `||`, `!`)
- Relational operators (`==`, `!=`, `<`, `>`, `<=`, `>=`)
- Nested conditionals
- Input validation basics
