# Week 6 — Multi-Dimensional Arrays

Two-dimensional and multi-dimensional arrays for tabular data and grid-based applications.

## Tasks

| # | File | Description |

## Concepts Covered

- Declaring and initializing 2D arrays
- Nested loops for row-column traversal
- Row-major vs column-major operations
- Matrix transposition
- 2D-to-1D mapping (linearization)
- Grid-based game logic
